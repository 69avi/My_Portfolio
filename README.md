# Priyanshu Sajwan — Portfolio

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Dark-themed with glassmorphism UI, smooth animations, and an Itachi Uchiha aesthetic.

🌐 **Live Site:** [69avi.github.io/My_Portfolio](https://69avi.github.io/My_Portfolio)

---

## Features

- Animated splash/loader screen
- Custom cursor with parallax background
- Hero & About sections with hover-play video (pauses on exact frame when cursor leaves)
- Animated skill bars (scroll-triggered)
- Featured projects with GitHub links and live demos
- Resume viewer modal with download option
- Contact form
- Fully responsive
